# MacroPad-HackClub

## Overview
This project is a custom macropad featuring a 3x4 keyboard, 1 rotary encoder and a 0.91 inch OLED display.

## BOM
- 11x Cherry MX Switches
- 11x Blank DSA Keycaps (White)
- 1x EC-11 Rotary Encoder
- 1x 0.91 inch OLED display
- 12x 1N4148 Diodes
- 2x 10kOhm Resistor
- 2x 1kOhm Resistor
- 2x 0.1uF Capacitor
- 1x XIAO RP2040
- 1x Case (2 printed parts)
